## 🚀 Getting Started

### Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- [Node.js](https://nodejs.org/) (optional, for local dev server)

### Run Locally

```bash
# Clone the repository
git clone https://github.com/ritesh-2124/portfolio.git

# Navigate to the project
cd portfolio

# Option 1: Open directly
open index.html

# Option 2: Use a local dev server
npx -y serve . -l 3000
```

Then visit **http://localhost:3000** in your browser.

## 🎨 Customization

Update the CSS variables in `css/style.css` to change the color scheme:

```css
:root {
    --accent-cyan: #00d4ff;
    --accent-purple: #7c3aed;
    --accent-pink: #ec4899;
    --accent-green: #10b981;
    --bg-primary: #0a0a0f;
}
```
